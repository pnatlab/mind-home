# Mind_Home

Mind_Home คือ web interface ขนาดเล็กที่ออกแบบมาเพื่อช่วยให้ผู้ใช้ค่อย ๆ กลับมาวางใจ เมื่อมีบางสิ่งเข้ามากระทบภายใน

โปรเจกต์นี้เป็นต้นแบบที่เน้นความเบา เรียบ และใช้ได้จริงใน browser โดยมีทั้งเวอร์ชันหลักที่นิ่งแล้ว และเวอร์ชันทดลองที่เริ่มฟัง “แรงของใจ” ได้ละเอียดขึ้น

## โครงสร้างไฟล์ในปัจจุบัน

ภายในโฟลเดอร์ Mind_Home มีไฟล์หลักดังนี้

### `index.html`
เวอร์ชันหลักของแอป

ใช้สำหรับ:
- การใช้งานทั่วไป
- การแชร์
- การทดลองใช้งานในรูปแบบที่นิ่งและเรียบง่าย
- flow แบบ reflective เดิมของระบบ
- การใช้งานสองภาษา ไทย / อังกฤษ

### `mind_home_intent_v2.html`
เวอร์ชันทดลองที่ต่อยอดจากของเดิม

เวอร์ชันนี้เพิ่มแนวคิดเรื่อง:
- การฟังเจตนาในข้อความ
- การตรวจจับ keyword ไทย / อังกฤษ
- การแยกเส้นทางของระบบตามแรงหลักในใจ เช่น
  - ความกลัว
  - ความอยาก
  - กรณีทั่วไป
- microcopy ที่ใกล้กับเสียงของหนูตื่นสายมากขึ้น
- แนวทาง interface ที่ได้รับแรงบันดาลใจจาก LLI / FLP / FLU

เวอร์ชันนี้เหมาะสำหรับการทดลอง ปรับปรุง และต่อยอดแนวคิดของระบบ

## จุดประสงค์หลัก

Mind_Home ไม่ได้ถูกออกแบบมาเพื่อเร่งให้เกิดคำตอบ

แต่มันถูกออกแบบมาเพื่อ:
- รับสิ่งที่กระทบใจเข้ามาก่อน
- ชะลอแรงตอบสนองทันที
- พาผู้ใช้เข้าสู่โหมดสังเกต
- ช่วยเปลี่ยนบริบทผ่านการกระทำเล็ก ๆ ทางกายภาพ
- ค่อย ๆ พากลับสู่สภาวะที่นิ่งขึ้น

ในเวอร์ชันทดลองใหม่ ระบบยังเริ่มทำสิ่งเหล่านี้เพิ่มขึ้นด้วย:
- ฟังว่าแรงหลักในข้อความเอนมาทาง “กลัว” หรือ “อยาก”
- ปรับน้ำเสียงของระบบให้สอดคล้องกับแรงนั้น
- ช่วยสร้างระยะระหว่างผู้ใช้กับแรงในใจ ก่อนจะรีบตอบสนอง

## โครงสร้างประสบการณ์

ระบบเวอร์ชันปัจจุบันใช้ลำดับหลักประมาณนี้

1. **Trigger Input**  
   วางสิ่งที่กำลังกระทบใจลงในระบบ

2. **Cooldown / Breathing Mode**  
   ให้เวลาระบบหายใจ และหยุดแรงตอบสนองอัตโนมัติ

3. **Read-Only Reflection**  
   เปลี่ยนจากผู้ตอบสนอง มาเป็นผู้สังเกตการณ์

4. **Soft Context Shift**  
   เลือกการกระทำทางกายภาพง่าย ๆ เพื่อออกจากลูปเดิม

5. **Return / Restart**  
   ปิดด้วยถ้อยคำ anchor และเปิดโอกาสให้เริ่มใหม่ได้เสมอ

ใน `mind_home_intent_v2.html` ลำดับนี้อาจมีน้ำเสียงต่างกันไปตาม route ที่ระบบตรวจพบ

## แนวทางการออกแบบ

Mind_Home ใช้แนวทางดังนี้
- ฟ้าอ่อน ขาว กระจกใส
- ความรู้สึกโปร่ง เบา และไม่กดดัน
- ภาษาแบบสงบ ละเมียด และมีระยะ
- ระบบสองภาษา ไทย / อังกฤษ
- interaction ที่ไม่เร่ง ไม่บีบ และไม่สั่งการเกินไป

ในเวอร์ชันทดลอง ระบบยังเริ่มสำรวจแนวคิดเพิ่ม เช่น
- intention-aware UX
- emotion-sensitive branching
- interface ที่ฟังก่อนตอบ
- human-AI interaction แบบที่ไม่รีบสรุปแทนผู้ใช้

## แนวทางการใช้งานที่แนะนำ

- ใช้ `index.html` เป็นเวอร์ชันหลักที่นิ่งแล้ว
- ใช้ `mind_home_intent_v2.html` เป็นเวอร์ชันทดลองสำหรับต่อยอด
- ถ้าเวอร์ชันทดลองนิ่งขึ้นในอนาคต ค่อยพิจารณาให้มาแทนเวอร์ชันหลัก

## สถานะของโปรเจกต์

Mind_Home ยังอยู่ในระยะ prototype  
ใช้เพื่อสำรวจแนวคิดเรื่อง:
- emotional transition interface
- reflective system design
- soft psychological firewall
- intention-driven human-AI co-creation
- การออกแบบระบบที่เริ่มฟัง “แรงในใจ” ได้มากขึ้น

## หมายเหตุ

Mind_Home ไม่ใช่เครื่องมือทางการแพทย์ และไม่ได้ออกแบบมาเพื่อทดแทนการดูแลทางสุขภาพจิตโดยผู้เชี่ยวชาญ

โปรเจกต์นี้เป็นต้นแบบเชิงออกแบบ  
เพื่อช่วยสร้างระยะ ความนิ่ง การรับรู้ตนเอง และการกลับมาสู่ภาวะที่เบาลงอย่างอ่อนโยน

มันเป็นส่วนหนึ่งของทิศทางที่ใหญ่กว่า ซึ่งเกี่ยวข้องกับ:
- system thinking
- emotional architecture
- reflective UX
- intention-aware interface
- human-centered AI collaboration
