# Mind_Home

Mind_Home is a soft reflective web interface designed to help a user pause, observe, and gently return to a calmer internal state.

This project is built as a lightweight single-file HTML application with a glass-inspired visual language and a step-based emotional reset flow.

## Core Purpose

Mind_Home is not designed to force a solution.

It is designed to:
- receive an emotional trigger
- create a short pause
- shift the user into observation mode
- guide a gentle physical context switch
- return the user to a more grounded state

## Experience Flow

The interface currently follows 4 main stages:

1. **Trigger Input**  
   The user places the current emotional disturbance into the system.

2. **Cooldown / Breathing Mode**  
   The interface creates a brief pause before any reaction.

3. **Read-Only Reflection**  
   The user is guided into observer mode instead of immediate response.

4. **Soft Context Shift**  
   The user chooses a simple physical action to help exit the emotional loop.

5. **Return / Restart**  
   The system closes with a gentle anchor and allows the cycle to begin again.

## Design Direction

Mind_Home uses:
- soft glassmorphism UI
- light blue / white reflective palette
- calm, minimal transitions
- bilingual interaction (Thai / English)
- microcopy designed to feel gentle, grounded, and non-reactive

## Current Version

Current version includes:
- single-file HTML/CSS/JS structure
- Thai / English language toggle
- glass UI in NuTuenSai-inspired tone
- local interaction logic without backend
- browser-ready prototype that can run without CodeSandbox

## Project Status

This project is currently in prototype stage.

It is being explored as:
- a reflective emotional interface
- a micro-ritual reset tool
- a psychological firewall / mind-state transition prototype
- an example of intention-driven human-AI co-creation

## Notes

Mind_Home is part of a broader design direction related to system thinking, emotional architecture, and human-centered reflective tools.

It is not intended as medical or therapeutic treatment.  
It is a personal design prototype for grounding, observation, and gentle self-regulation.

# Mind_Home

Mind_Home คือ web interface ขนาดเล็กที่ออกแบบมาเพื่อช่วยให้ผู้ใช้ค่อย ๆ กลับมาวางใจ เมื่อมีบางสิ่งเข้ามากระทบภายใน

โปรเจกต์นี้เป็นต้นแบบแบบ single-file HTML ที่รวมโครงสร้าง HTML, CSS และ JavaScript ไว้ในไฟล์เดียว เพื่อให้สามารถรันได้ง่ายใน browser และใช้เป็นพื้นที่ทดลองแนวคิดด้าน emotional interface, reflective UX และ human-centered system design

## จุดประสงค์หลัก

Mind_Home ไม่ได้ถูกออกแบบมาเพื่อเร่งให้เกิดคำตอบ

แต่มันถูกออกแบบมาเพื่อ:
- รับสิ่งที่กระทบใจเข้ามาก่อน
- ชะลอแรงตอบสนองทันที
- พาผู้ใช้เข้าสู่โหมดสังเกต
- ช่วยเปลี่ยนบริบทผ่านการกระทำเล็ก ๆ ทางกายภาพ
- ค่อย ๆ พากลับสู่สภาวะที่นิ่งขึ้น

## โครงสร้างประสบการณ์

ระบบเวอร์ชันปัจจุบันประกอบด้วยลำดับหลักดังนี้

1. **Trigger Input**  
   วางสิ่งที่กำลังกระทบใจลงในระบบ

2. **Cooldown / Breathing Mode**  
   ให้เวลาระบบหายใจและหยุดแรงตอบสนองอัตโนมัติ

3. **Read-Only Reflection**  
   เปลี่ยนจากผู้ตอบสนอง มาเป็นผู้สังเกตการณ์

4. **Soft Context Shift**  
   เลือกการกระทำทางกายภาพง่าย ๆ เพื่อออกจากลูปเดิม

5. **Return / Restart**  
   ปิดด้วยถ้อยคำ anchor และเปิดโอกาสให้เริ่มใหม่ได้เสมอ
