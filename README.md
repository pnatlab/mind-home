# Mind_Home

Mind_Home is a soft reflective web interface designed to help a user pause, observe, and gently return to a calmer internal state.

This project is built as a lightweight browser-based prototype with a glass-inspired visual language, bilingual interaction, and a step-based emotional reset flow.

It currently includes both a stable baseline version and a more intention-aware experimental version.

## Files

### `index.html`
Stable version of Mind_Home.

This version is designed for:
- regular use
- sharing
- a calm default reflective flow
- bilingual Thai / English interaction
- glass-inspired emotional reset experience

### `mind_home_intent_v2.html`
Experimental version of Mind_Home.

This version extends the original flow by adding:
- intention-aware input routing
- keyword detection in Thai and English
- route separation between:
  - fear
  - desire
  - default
- NuTuenSai-style listening microcopy
- a softer interface layer inspired by LLI / FLP / FLU logic

This version is intended for testing, refinement, and deeper system design exploration.

## Core Purpose

Mind_Home is not designed to force a solution.

It is designed to:
- receive an emotional trigger
- create a pause before reaction
- help the user move into observation mode
- guide a gentle physical context shift
- return the user to a more grounded internal state

In the newer experimental direction, it also begins to:
- listen for the emotional direction inside the text
- distinguish between fear-driven and desire-driven movement
- adapt the tone of the interface before the user reacts

## Experience Flow

The current interface follows a reflective sequence such as:

1. **Trigger Input**  
   The user places what is pressing on the mind into the system.

2. **Cooldown / Breathing Mode**  
   The interface creates a short pause before immediate reaction.

3. **Read-Only Reflection**  
   The user is shifted from reacting to observing.

4. **Soft Context Shift**  
   The user chooses a small physical action to step out of the current mental loop.

5. **Return / Restart**  
   The system closes with a gentle anchor and allows the cycle to begin again.

In `mind_home_intent_v2.html`, this flow can vary depending on detected emotional direction.
